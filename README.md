
# 🔙 Zerodha Backend

> Node.js & Express backend for Zerodha Clone Trading Platform.

---

## 🚀 Features

* 🔐 JWT Authentication
* 📡 REST APIs
* 💼 Portfolio Management
* 💰 Buy/Sell Trading Logic
* 🗄️ MongoDB Integration

---

## 🛠️ Tech Stack

* Node.js
* Express.js
* MongoDB
* JWT Authentication

---

## ⚙️ Setup Instructions

```bash
git clone https://github.com/Harshjadhav003/zerodha-backend.git
cd zerodha-backend
npm install
npm start
```

Server runs on:
👉 http://localhost:3002

---

## 📡 API Endpoints

### Auth

* POST /auth/signup
* POST/auth/login

### Trading

* GET /allOrders
* GET /allPositions


### Portfolio

* POST /allHoldings

## 🗄️ Database
